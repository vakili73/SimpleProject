برای استفاده از جستجوی هوشمند نیاز هست تا کمی دست به کیبورد بشید، چون قراره باهاش کارای باحال زیادی بکنیم. ;-)

خب! اول نیاز هست که با چند علامت و کاربرد آنها در جستجو آشنا بشید، بریم!

; -> [ ; ] از این علامت برای جدا سازی در بین شروط استفاده میشود

: -> [ Shift + ; ] برای ایجاد یک دامنه از این علامت استفاده میکنیم

برای فهم بهتر به مثال های زیر دقت کنید

فیلتر ساده:

مثال(پله): 2580
گزارش شامل تمامی رکوردها در پله ی 2580 است

مثال(بلاست): 78:143
رکوردهای گزارش بین بلاست 78 تا 143 را برمیگرداند

فیلتر چند تایی:

مثال(عیار): 0.2;0.5:0.8
شمامل تمامی رکوردهایی میشود که عیار آنها 0.2 یا در بازه 0.5 تا 0.8 میباشد هست

مثال(تناژ): 120;80;40:60;140:180
خب دیگه! خودتون حدس بزنید چکار میکنه ... ;-)
